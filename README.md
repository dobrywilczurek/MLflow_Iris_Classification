Projekt demonstruje wykorzystanie MLflow do śledzenia eksperymentów machine learning. 
Klasyfikujemy gatunki irysów (Iris flower dataset) za pomocą drzew decyzyjnych, logując wszystkie parametry, metryki i modele.

MLflow - platforma do zarządzania cyklem życia ML

scikit-learn - biblioteka do uczenia maszynowego

Iris dataset - zbiór danych do klasyfikacji
